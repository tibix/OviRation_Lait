# OviRation Lait

**Logiciel SaaS de calcul nutritionnel ovin laitier — Agrilium**

OviRation Lait est un outil professionnel de rationnement ovin basé sur les normes **INRA 2007**, conçu par un éleveur pour les éleveurs. Il se compose de deux interfaces complémentaires :

- **OviRation Lait PC** — Calcul et configuration des rations par lot, gestion sanitaire (Piétin, Visna-Maedi), synchronisation cloud
- **OviRation Lait Mobile** — Distribution journalière, suivi des refus, saisie de la production laitière

🌐 **Application disponible sur :** [myagrilium.com](https://www.myagrilium.com)
